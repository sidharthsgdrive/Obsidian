## Landing Page
Sections:
- *My Surveys* section, displaying a list of links for surveys that the user has created
- *Create Survey* button (Brings up Create Survey page)
- *Answer Survey* (button or text field with a button): takes a survey code as input, and then redirects to a page that allows to answer the survey

## Create Survey Page
- prompt for number of questions
- form that asks to enter text for each question

## Answer Survey Page
- Remember, the kind of survey we are going with is an *opinion poll*, which consists of 5 options for each question: strongly disagree, disagree, neutral, agree, and strongly agree. 
- So there will be each question displayed, with 5 radio buttons (nicely styled like in the image attached)
- And a submit button

## Survey Analysis Page
- this is the page that has to be created for each survey, and is viewable only by the survey creator.
- There will be sections for each question, displaying the average response (as an integer score from -2 to +2, as well as the most closely matching response option)
