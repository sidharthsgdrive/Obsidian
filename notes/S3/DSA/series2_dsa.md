linked list, tree, heap, graph
note: binary search tree sorting is not there for this exam.

- Linked List
	- concept of linked list
	- implementation: `createnode()`, `insertfront()`i, `insertback()`, `traverse()`, `deletelast()`, `deletefirst()`, `insert()`
	- stack using linked list
	- queue using linked list
- Binary Tree
	- concepts: Ancestors, Descendants, Path, Height, Level, Degree
	- skewed binary tree, complete binary tree
	- sequential representation of binary tree
	- traversal of linked binary tree: pre-order, in-order, post-order, level-order
	- Binary Search Tree (BST)
		- ~~BST sort~~ (not there for this exam)
		- implementation: insertion, deletion
		- balanced BST
	- Expression Tree
- Heap
	- Properties
	- implementation: `heapify`, `insert` , `delete`, `build` (for both min and max heap)
	- Heap Sort
	- Priority Queue using heap
		- `insert`, `delete`, `findmax`, `decreasekey`
	- counting number of heaps
- Graph
	- directed and undirected graphs
	- maximum and minimum edge count
	- representation
		- Adjacency Matrix
		- Adjacency List
	- traversal
		- DFS (recursive and non-recursive)
		- BFS
