++1. **S**OLID++
- Each class must ideally perform only one function.
++2. **O**PEN/CLOSED++
- Classes should stay *closed* to alteration, but should be possible to extend via inheritance, overriding, etc.
++3. **L**ISKOV SUBSTITUTION++
- Derived classes must be interchangeable with base classes, i.e: must support all functionality of base class.
++4. **I**NTERFACE SEGREGATION++
- A class should never be required to implement an interface or rely on a method that it doesn't use.
++5. **D**EPENDENCY INVERSION++
- We should rely on abstractions such as interfaces and abstract classes, that are dependent on details, rather than details being dependent on abstractions.
