When creating nested subdirectories, use -p flag. 
``badsfsh
mkdir -p src/{module1,module2}
```

## Creating links

Hard links:
```bash
ln path/to/destination path/to/linkname
```

Soft links:
```bash
ln -s path/to/destination path/to/linkname
```

Note: destination paths are relative to the directory that the link is located in.
