* [remotely-save plugin](https://github.com/remotely-save/remotely-save)
	* seems to be backed by some reddit users as reliable
	* possibly less of a hassle to sync on mobile than the current method, have to look into it
	* update -- google drive support is a paid feature
* [using Git](https://viscouspotential.medium.com/how-to-sync-obsidian-across-all-devices-using-git-automatically-and-for-free-dd3c76e7447b)
	* harder to set up
	* supports version control
	* possibly more robust
* [Obsidian Decentralized plugin](https://github.com/iWebbIO/obsidian-decentralized/) 
	* available on Github, not yet released on community plugins
	* syncs over P2P network
	* sounds cool and might be more performant
	* also hard to set up, probably
* [SyncThing](https://syncthing.net/)
	* Tried and tested, popular
	* Mobile app is a fork
* [obsidian-git](https://github.com/Vinzent03/obsidian-git)
	* look into this
