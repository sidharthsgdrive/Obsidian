### Thoughts

I think I figured out my network issue when playing minecraft.
I believe the main problem was that chunks weren't being able to be sent to the client from the server fast enough, so it would kick me because was putting too much load on the server. 
So I reduced the maximum server render distance from 16 to 12 (I think the reason I was being disproportionately affected was because I usually kept a higher render distance than the others)
Also, it might lag more when only one person is online (counterintuitively, as one would expect the server to lag more with more people online), since if other people are online, they will keep chunks loaded and hence the server won't have to reload them when a player comes to that area.
